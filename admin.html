How It Works
📧 Auto Receipt Email
  • Triggers when admin adds a contribution
  • Sent to that member's email only
  • Includes receipt number, amount, date
  • Skipped if daily email quota is full
📅 Monthly Report Email
  • Runs on last day of the month (Apps Script trigger)
  • Contributed → receives their payment summary
  • Not contributed → receives a reminder email
  • Each member gets only their own data
Monthly report requires a Time-based trigger set up in Apps Script. Go to Apps Script → Triggers → Add trigger → sendMonthlyEndReports → Month timer → End of month.
Manual Monthly Report (Test)
Send the current month's report to all members right now (for testing or immediate dispatch).
