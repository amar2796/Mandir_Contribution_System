Automatically sends a receipt email to the member right after a contribution is saved. Only sent if member has an email address.

Monthly End-of-Month Report
On the last day of each month, sends each member a personal email — receipt summary if they contributed, or a friendly reminder if they haven't.
Loading status...
How It Works
📧 Auto Receipt Email
  • Triggers when admin adds a contribution
  • Sent to that member's email only
  • Includes receipt number, amount, date
  • Skipped if daily email quota is full
📅 Monthly Report Email
  • Runs on last day of the month (Apps Script trigger)
  • Contributed → receives their payment summary
  • Not contributed → receives a reminder email
  • Each member gets only their own data
Monthly report requires a Time-based trigger set up in Apps Script. Go to Apps Script → Triggers → Add trigger → sendMonthlyEndReports → Month timer → End of month.
Receipt Email Logo URL
Paste the publicly accessible image URL for the temple logo used in all email receipts. Host on Google Drive → share publicly → use lh3.googleusercontent.com/d/FILE_ID=w400 format. Leave blank to use the 🕉️ symbol fallback.
